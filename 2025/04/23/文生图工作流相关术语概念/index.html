文生图工作流相关术语概念

Tmerowr2025-04-232025-04-23

总结一下 ComfyUI 中常见的名词，特别是与文生图（Text-to-Image）工作流相关的术语和概念。
这些名词是理解和操作 ComfyUI 的基础，涵盖节点、模型、参数等方面。

节点相关名词

Node (节点)：ComfyUI 的基本构建单元，每个节点代表一个功能（例如加载模型、编码文本、采样等）。用户通过连接节点构建工作流。
Workflow (工作流)：由多个节点连接组成的一个完整流程，例如从文本生成图像的工作流。
Input (输入)：节点接收的数据，例如提示词、模型文件等。
Output (输出)：节点生成的结果，例如编码后的条件、生成的图像等。

模型相关名词

Checkpoint (检查点)：预训练的 Stable Diffusion 模型文件（.ckpt 或 .safetensors 格式），包含模型权重，例如 SD1.5、SDXL 等。
CLIP：用于处理文本的模型模块，将提示词编码为模型可理解的条件。
VAE (变分自编码器)：用于将潜在图像（Latent Image）编码或解码为实际图像的模块。
LoRA (Low-Rank Adaptation)：一种轻量级微调模型，用于在不改变原始模型的情况下添加特定风格或主题（例如卡通风格）。
ControlNet：一种模型扩展，用于通过附加条件（如姿势、边缘图）控制生成结果。

提示词相关名词

Prompt (提示词)：描述生成内容的文本输入，例如“a cute cat in a sunny garden”。
Positive Prompt (正向提示词)：描述你想要生成的内容。
Negative Prompt (负向提示词)：描述你不希望出现在生成结果中的内容，
例如“blurry, low quality”。
Conditioning (条件)：提示词经过 CLIP 编码后生成的条件，供采样器使用。

图像与潜在空间相关名词

Latent Image (潜在图像)：图像在潜在空间中的表示形式，是扩散模型处理的对象，比实际像素图像更小，节省计算资源。
Image (图像)：最终生成的像素图像，通常由 VAE Decode 解码得到。
Resolution (分辨率)：生成图像的尺寸，例如 512x512、1024x768。
Batch Size (批次大小)：一次生成图像的数量，例如生成 1 张或多张。

采样相关名词

Sampler (采样器)：扩散模型生成图像时使用的算法，例如 Euler a、DPM++ 2M Karras、DDIM 等。
Scheduler (调度器)：控制采样过程中的步进策略，例如 Karras、Normal。
Steps (步数)：采样过程中的迭代次数，通常 20-50 步，步数越多细节越精细。
CFG Scale (提示词引导系数)：控制提示词对生成结果的影响程度，通常 7-12。
Seed (种子)：随机种子，用于控制生成结果的随机性，固定种子可复现相同结果。

文件与路径相关名词

Checkpoint File (模型文件)：Stable Diffusion 模型的存储文件，通常放在 ComfyUI 的 models/checkpoints 文件夹中。
Output Directory (输出目录)：生成图像保存的文件夹，通常是 ComfyUI 的 output 文件夹。
Filename Prefix (文件名前缀)：保存图像时指定的前缀，例如“output”。

扩展功能相关名词

Inpainting (修补)：对图像的特定区域进行修改或生成，例如修复缺失部分。
Outpainting (扩展)：扩展图像边界，生成更大的图像。
Upscale (放大)：提升图像分辨率，例如从 512x512 放大到 1024x1024。
AnimateDiff：一个扩展，用于生成动画或视频。

其他常见名词

Stable Diffusion (稳定扩散)：ComfyUI 使用的核心扩散模型，用于从噪声生成图像。
Diffusion Process (扩散过程)：从随机噪声逐步生成图像的过程，核心是去噪。
Community (社区)：ComfyUI 的用户群体，提供教程、自定义节点等资源，例如 GitHub 上的 ComfyUI 社区。
Custom Node (自定义节点)：社区或用户开发的额外节点，扩展 ComfyUI 功能。

总结

这些名词涵盖了 ComfyUI 文生图工作流的核心概念：
基础：Node、Workflow、Input、Output。
模型：Checkpoint、CLIP、VAE、LoRA、ControlNet。
提示词：Prompt、Positive/Negative Prompt、Conditioning。
图像：Latent Image、Image、Resolution、Batch Size。
采样：Sampler、Scheduler、Steps、CFG Scale、Seed。
文件：Checkpoint File、Output Directory、Filename Prefix。
扩展：Inpainting、Outpainting、Upscale、AnimateDiff。
